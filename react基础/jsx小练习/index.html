注意区分 js语句（代码） 与 js表达式 的区别

拿个变量在左侧接，能接到的就是表达式

  1. 表达式：一个表达式会产生一个值，可以存放在任何一个需要值的地方。下面这些都是表达式：
    1. a （变量名）
    2. a+b
    3. demo(1) // demo函数没写返回值也会有默认的返回值为undefind
    4. arr.map()
    5. function test() {} // 定义函数也有一个返回值

  2. 语句（代码） 下面这些都是

    1. if(){}
    2. for(){}
    3. switch(){case:xxx}
